fmpro
=========

- FM player based on DOUAudioStreamer

### Features

- 基本播放流程
- Basic music player
- 歌词
- Support lyrics
- 后台播放和控制
- Background music and player controls
- 锁屏后封面+歌词
- Support display cover & display lyrics on lock screen
